# Clipboard Image Sidebar (クリップボード画像管理エキステンション)

画像をクリップボードからサイドバーに保存し、いつでも再利用できるようにする Chrome 拡張機能です。スクリーンショット、タグ管理、ドラッグ＆ドロップなどの機能を備えています。

## ✨ 主な機能

- **画像の自動保存**: クリップボードにコピーした画像をサイドバーにペースト (`Cmd+V`) するだけで保存。
- **ドラッグ＆ドロップ**: Finder やブラウザから画像を直接サイドバーにドロップして保存。
- **スクリーンショット**: 現在のタブを 1 クリック、またはショートカットでキャプチャ。
- **タグ管理**: 保存した画像にタグを付けて分類・フィルタリング。
- **再利用**: 保存した画像を 1 クリックでクリップボードにコピー、またはデスクトップにダウンロード。
- **ブラウザ内ペースト**: サイドバーの画像をブラウザの入力欄にドラッグ＆ドロップして貼り付け。
- **永続保存**: IndexedDB を使用し、ブラウザを閉じても画像データをローカルに保持。

## ⌨️ キーボードショートカット

| ショートカット | 機能 |
|--------------|------|
| **`Cmd + Shift + S`** (Mac) / **`Ctrl + Shift + S`** (Win) | 現在のタブをスクリーンショットして保存 |
| **`Cmd + Shift + C`** (Mac) / **`Ctrl + Shift + C`** (Win) | サイドパネルを開く |
| **`Cmd + V`** (Mac) / **`Ctrl + V`** (Win) | クリップボードから画像をペースト保存 (サイドバーにフォーカス中) |

## 🛠 インストール方法

1. このプロジェクトのフォルダをダウンロードまたはクローンします。
2. Chrome ブラウザを開き、アドレスバーに `chrome://extensions/` を入力します。
3. 右上の「デベロッパー モード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、このプロジェクトのフォルダを選択します。

## 📁 ファイル構成

```text
clipboard-image-sidebar/
├── manifest.json      # 拡張機能の設定ファイル
├── background.js       # バックグラウンド処理（ショートカット、スクショ）
├── sidepanel.html      # サイドバーの構造
├── sidepanel.css       # サイドバーのデザイン
├── sidepanel.js        # サイドバーのロジック・IndexedDB管理

    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

## 🧰 テクノロジー

- **Manifest V3**: 最新の Chrome 拡張機能規格。
- **Side Panel API**: ブラウザの横に常駐する UI を提供。
- **IndexedDB**: 画像データ（DataURL）をローカルに高速保存。
- **Clipboard API**: クリップボードの読み書きを制御。
- **Tabs API**: アクティブタブのキャプチャを処理。
