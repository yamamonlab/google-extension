# Google Extension Collection

Google Chrome 拡張機能のコレクションリポジトリです。
日々の業務やブラウジングを効率化するツールを公開しています。

## 📁 収録されている拡張機能

### 1. [Clipboard Image Sidebar](./clipboard-image-sidebar/)
画像をクリップボードからサイドバーに保存し、管理・再利用できるツール。
- **機能**: スクショ撮影、タグ管理、ドラッグ＆ドロップ、IndexedDBによる永続化。
- **使い方**: `Cmd + Shift + S` でスクショ撮影。サイドパネルでタグ付けやコピーが可能。

---

## 🛠 インストール方法

1. このリポジトリをダウンロードまたはクローンします。
2. 各プロジェクトフォルダ（例: `clipboard-image-sidebar/`）内の手順に従って Chrome に読み込ませてください。
